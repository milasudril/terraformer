\[
 d^2(t, P) = |f(t) - P|^2

\[
\frac{\mathrm{d}}{\mathrm{d} t}\left(d^2(t, P_0)\right) =
\frac{\mathrm{d}}{\mathrm{d} t}\left|P(t) - P_0\right|^2 =
2\left(P - P_0\right)\cdot\dot{P}  = 0
\]

Let $P(t) = O + \vec{C}t + \vec{B}t^2 + \vec{A}t^3$. Then $\dot P (t) = \vec{C} + 2\vec{B}t + 3\vec{A}t^2$.
